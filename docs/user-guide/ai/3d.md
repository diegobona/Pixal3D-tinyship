# 3D Generation

Pixal3D converts an input image into a textured GLB model. The primary provider is fal.ai's `fal-ai/pixal3d` endpoint. Wiro's TencentARC Pixal3D endpoint remains available as a backup provider.

To enable real generation, configure:

```env
FAL_API_KEY="your-fal-api-key"
```

To use the Wiro backup provider, configure:

```env
WIRO_API_KEY="your-wiro-api-key"
WIRO_API_SECRET="your-wiro-api-secret"
```

Generation consumes site credits before the provider call starts. Credits are refunded automatically only if task creation fails before fal/Wiro accepts the request, for example a missing API key or submit-time validation error. If the provider accepts the request and later reports a runtime failure, the task is marked failed without an automatic site-credit refund because the upstream provider may already have billed the request.
